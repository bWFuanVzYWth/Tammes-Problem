# Tammes-Problem

## 简介

在球面上排列点，使点之间的最小距离最大化  
为了优化游戏戴森球计划中的蓝图写的，这个算法不具备学术上的严谨性，但是在求近似解的问题上拥有极高的性能  
对单次迭代，它关于点数N的时间复杂度为O(logN)，因此能优化上万规模的问题
